Image Representation and Fundamentals

Understanding how digital images are represented and processed

What is an Image?

An image is a two-dimensional function representing light intensity at each spatial position. In digital form, an image is represented by a grid of pixels, where each pixel holds numerical values corresponding to brightness or color.

Types of Digital Images

Digital images can be classified into various types based on their content and representation:

  • Grayscale Images: Each pixel contains a single intensity value, typically ranging from 0 (black) to 255 (white).
  • Binary Images: Pixels are represented by 0 or 1, used for segmentation and pattern recognition.
  • RGB (Color) Images: Each pixel has three values (Red, Green, Blue) to define color.
  • Indexed Images: Uses a colormap where pixel values correspond to specific colors in a predefined palette.

Image Resolution

Resolution refers to the amount of detail in an image, defined by:

  • Spatial Resolution: Number of pixels in width and height (e.g., 1920x1080).
  • Bit Depth: The number of bits used per pixel to represent intensity levels (e.g., 8-bit, 16-bit).
  • Temporal Resolution: Applies to video frames, defining the frame rate (e.g., 30 fps).

Color Spaces

Color spaces define how colors are represented numerically. Some common color spaces include:

  • RGB: A combination of Red, Green, and Blue components.
  • HSV (Hue, Saturation, Value): Separates color information from intensity.
  • CMYK: Used in printing, based on Cyan, Magenta, Yellow, and Black.
  • Grayscale: A single-channel representation based on brightness.

Image File Formats

Different image formats are used for various applications based on compression, quality, and compatibility:

  • JPEG: Lossy compression, suitable for photographs.
  • PNG: Lossless compression, suitable for high-quality graphics.
  • BMP: Uncompressed format, preserving pixel data.
  • DICOM: Used in medical imaging, stores metadata alongside pixel data.

Image Representation in Programming

In programming environments, images are typically represented as multi-dimensional arrays. For example, in Python using NumPy:

MATLAB
import cv2
import numpy as np

# Load an image in grayscale
image = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# Get image dimensions
height, width = image.shape
print(f"Height: {height}, Width: {width}")

# Display the image
cv2.imshow('Grayscale Image', image)

Image Processing Fundamentals

Basic operations applied to digital images include:

  • Filtering: Smoothing, sharpening, and noise reduction.
  • Transformation: Rotation, scaling, and translation.
  • Segmentation: Partitioning an image into meaningful regions.
